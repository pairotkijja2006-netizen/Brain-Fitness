# 🧠 Brain Fitness Application

A Unity-based brain fitness application designed to help delay the onset of dementia through three memory-training games that engage both the left and right brain. The project was deployed at My Home Care Center in Bangkok, Thailand, where I collaborated with caretakers to introduce the game to elderly users.

---

## ▶️ Play the Game
👉 [Brain Fitness](https://jiratip.itch.io/brain-fitness)
*(Note: This application was designed specifically for elderly users in Thailand, so all in-game text, instructions, and titles are in Thai to ensure accessibility and ease of use for the target audience.)*
*(Best experienced on a tablet. Initial load may take a few seconds.)*

---

## 🕹️ How to Use the Application

*(Note: The game interface is in Thai, as it was designed for elderly users in Thailand.)*

### Walkthrough
1. **Start Screen**  
   - Click anywhere on the title screen to begin.

2. **Create / Select Profile**  
   - Enter a name to create a profile.  
   - Click the profile box to enter.  
   - Click the red button to delete a profile.

3. **Main Menu**  
   You will see three games:

   - 🎲 **Counting Dice Game**  
     Add or subtract the total number of points on the dice  
     → focuses on logical and numerical skills  

   - 🧩 **Shape Building Game**  
     Arrange shapes to match the target design  
     → trains visual and spatial (right-brain) skills  

   - 💰 **Money Counting Game**  
     Add different amounts of money together  
     → reinforces everyday numerical reasoning  

4. **Statistics (Top Left)**  
   - View progress over time through a graph  
   - Tracks improvement in memory and performance  

5. **Profile (Top Right)**  
   - Click to return to the profile selection screen

## 🎮 Features
- 🧩 Three interactive memory-training games  
- 🔁 Streak-based progression system to encourage consistency  
- 🏆 Reward system (win points) to motivate users  
- 👥 Multi-user support with account tracking  
- 📊 Leaderboard to compare performance  

---

## 🧠 Purpose
This application was created to explore how gamified cognitive exercises can support brain health and potentially help delay the onset of dementia. By combining engaging gameplay with memory-based challenges, the goal is to make brain training both accessible and enjoyable for elderly users.
